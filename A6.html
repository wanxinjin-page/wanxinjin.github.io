Image mosaic algorithm

Designed a fast image processing algorithm to achieve image mosaic which was used on aircraft. Control and Simulation Center at HIT, Undergraduate researcher, Research advisor: Xiaoguang Di, Project designer: the Military. (See video demo in the attachment)

Highlights

  • A modified Harris feature point extraction is proposed to avoid extracted feature points concentrating in one region of image. Descriptor of Harris feature point is presented to make the inter-frame feature point matching more efficient. RANSAC algorithm is then applied to remove the error matching points.
  • This image mosaic algorithm is suitable for a variety of forms of inter-frame motion including translation, rotation, affine transform and even small projection transform. The project is successfully completed with satisfied performance evaluation results.
